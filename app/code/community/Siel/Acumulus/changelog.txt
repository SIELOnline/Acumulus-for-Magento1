Changelog Acumulus webkoppelingen
=================================

Library die door iedere webwinkelkoppeling gebruikt wordt
=========================================================

Library 3.3.5
-------------
* Instellingen 'Debug' en 'Lokaal' zijn samengevoegd en kan via het
  instellingenformulier door gebruikers aangepast worden.
* Test/lokaal optie geïntegreerd in de normale code (omwille van een betere
  ondersteuning).
* Optie ingebouwd om te kiezen tussen bereiken van een bepaalde status of  het
  moment van aanmaken van de factuur (vooralsnog alleen gebruikt door Magento).
* Check ingebouwd op niet leeg klant-emailadres.

Library 3.3.4
-------------
* Optie toegevoegd om klantgegevens NIET naar Acumulus te sturen.

Library 3.3.3
-------------
* https in connector data
* 2 gebruikerinterface-teksten.

Library 3.3.2
-------------
* Country code voor landen buiten de EU moet leeg zijn, landnaam dient dan in
  veld city meegestuurd te worden.
* Veranderingen in connector gegevens die meegestuurd worden.

Library 3.3.1
-------------
Vanaf deze versie wordt de changelog bijgehouden.




Magento
=======
Geteste Magento versie(s):
* 1.9.0.1
* 1.8.1.0
* 1.7.? (getest door klanten, er worden nog wat mogelijke problemen onderzocht)

Magento-1.9-Acumulus-3.1.0
-----------------------------------
* Licentie teksten en deze changelog staan nu bij de code, geen zip in zip meer.
* Keuze toegevoegd om factuur automatisch te versturen als in Magento de factuur
  aangemaakt wordt.
* Facturen konden dubbel verstuurd worden als voor de "status voor verzending"
  'processing' is geselecteerd.
* Support en debug optie toegevoegd aan instellingenscherm.
* Handmatig (her)verzenden toegevoegd.
* BTW berekeningen gingen fout als catalogusprijzen ex. BTW worden opgegeven.
* Betalingskosten werden niet meegestuurd naar Acumulus.
* Library 3.3.5 (zie bij Library).

Magento-1.9-Acumulus-3.0.0
-----------------------------------
* Initiële versie van deze volledig herbouwde koppeling.
* Library 3.3.4 (zie bij Library).
